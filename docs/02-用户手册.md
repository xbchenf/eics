# EICS 企业智能客服系统 — 用户手册

> 版本：V2.0 | 最后更新：2026-06-28

---

## 1. 系统简介

EICS 是企业级智能客服平台，提供以下核心能力：

- **智能问答**：基于知识库的 AI 自动回复
- **工单提交**：多轮对话引导式报修/申请
- **人工服务**：一键转接人工坐席实时沟通
- **服务评价**：对人工服务进行满意度评分

---

## 2. 快速开始

### 2.1 注册账号

1. 浏览器打开系统地址（由管理员提供）
2. 点击「没有账号？立即注册」
3. 填写用户名、密码、姓名
4. 点击注册，自动登录进入聊天页面

### 2.2 登录

1. 打开系统地址
2. 输入用户名和密码
3. 点击登录，进入智能客服对话页面

---

## 3. 智能客服对话

```
├─────────────────────────────────────┤

├─────────────────────────────────────┤

├─────────────────────────────────────┤
│  [输入问题...]              [发送]   │  ← 输入区

```

### 3.2 智能问答

直接在输入框输入问题，系统会自动从知识库检索答案。支持的提问方式：

- "公司年假怎么申请？"
- "邮箱怎么配置？"
- "打印机怎么安装？"

### 3.3 提交工单

**方式一：点击快捷按钮**

点击输入框下方的「📋 提交工单」，按提示逐步填写：

1. 输入手机号（方便工作人员联系您）
2. 选择问题类型（硬件故障 / 网络问题 / 权限申请 / 软件安装 / 其他）
3. 描述故障详情
4. 提交成功 → 获得工单编号

**方式二：直接发送**

在输入框输入「提交工单」或「报修」，系统会自动启动工单提交流程。

### 3.4 转人工服务

**方式一**：点击「🔁 转人工」按钮

**方式二**：输入「转人工」或「人工」

系统会为您转接在线坐席，等待期间请勿关闭页面。接通后可与坐席实时文字沟通。

### 3.5 服务评价

人工服务结束后，系统会弹出评价卡片：

```
⭐ 请为本次服务评分
★★★★☆
[说说您的感受...]
[提交评价]
```

点击星星选择评分（1-5 星），可选填评价留言，点击提交。

> **提示**：如果当时未提交评价，下次打开聊天页面时，「📝 我的工单」按钮上会有红色角标提示。

## 4. 我的工单

点击顶栏「📝 我的工单」按钮，弹窗显示您提交过的所有工单：

| 信息 | 说明 |
|------|------|
| 问题类型 | 硬件故障 / 网络问题 / 权限申请等 |
| 工单状态 | 待处理 → 处理中 → 已解决 → 已关闭 |
| 工单编号 | 唯一标识，可用于查询进度 |
| 提交时间 | 工单创建时间 |

如果有待评价的服务，弹窗顶部会显示「⭐ 待评价服务」区域，可直接在弹窗内完成评分。

## 6. 常见问题

| 问题 | 解决 |
|------|------|
| 发消息没回复 | 检查网络连接，刷新页面重试 |
| 工单提交失败 | 确认手机号格式正确（11 位数字） |
| 转人工一直等待 | 当前坐席可能全部离线，请稍后再试或先提交工单 |
| 看不到历史消息 | 刷新页面会自动加载最近会话记录 |
| 页面显示异常 | 清除浏览器缓存后重试，推荐使用 Chrome 或 Edge |
