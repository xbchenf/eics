# EICS 企业智能客服系统 — 管理员手册

> 版本：V2.0 | 最后更新：2026-06-28

---

## 1. 概述

管理员拥有系统的全部管理权限，包括：坐席账号管理、工单管理、知识库维护、系统配置。

访问方式：使用管理员账号登录后自动进入管理后台。

---

## 2. 工作台总览

登录后首先看到的页面，展示系统运行概况：

| 指标 | 说明 |
|------|------|
| 今日会话 | 当天用户咨询总次数 |
| 待处理工单 | 当前状态为「待处理」的工单数 |
| 在线坐席 | 当前在线的人工坐席数 |
| 知识库文档 | 已上传的文档总数 |
| 满意度 | 所有评价的平均分和总数 |
| 今日工单 | 当天创建的工单数 |

页面还包含：
- **优先级分布图**：待处理工单的 P0/P1/P2/P3 分布
- **7 日趋势图**：近一周会话和工单的数量趋势

> 数据每 30 秒自动刷新。

## 3. 用户管理

入口：**系统设置 → 用户管理**

### 3.1 创建用户

1. 点击「新建用户」
2. 填写用户名、密码、姓名
3. 选择角色：USER（普通用户）/ AGENT（坐席）/ ADMIN（管理员）
4. 点击确认

### 3.2 管理用户

| 操作 | 说明 |
|------|------|
| 启用/禁用 | 切换用户状态，禁用后无法登录 |
| 修改角色 | 可随时调整用户角色 |

### 3.3 权限说明

| 角色 | 权限 |
|------|------|
| USER | 聊天咨询、提交工单、查看自己的工单、服务评价 |
| AGENT | 接入会话、处理工单、使用快捷回复、查看满意度 |
| ADMIN | 全部权限：用户管理、知识库管理、系统配置、所有数据查看 |

## 4. 工单管理

入口：**工单管理**

### 4.1 工单列表

展示所有工单，支持按状态筛选和关键词搜索。

| 列 | 说明 |
|------|------|
| 工单号 | 唯一标识，点击进入详情 |
| 电话 | 用户手机号（中间四位脱敏） |
| 问题类型 | 硬件故障 / 网络问题 / 权限申请 / 软件安装 / 其他 |
| 优先级 | P0（红色）/ P1（橙色）/ P2（蓝色）/ P3（灰色） |
| 状态 | 待处理 → 处理中 → 已解决 → 已关闭 |
| SLA | 截止时间及剩余/超时分钟数 |

### 4.2 工单处理流程

```
用户提交工单 → 状态: PENDING（待处理）
     │
     ├─ 坐席点击「认领」→ 状态: PROCESSING（处理中）
     │
     └─ 坐席点击「解决」→ 状态: RESOLVED（已解决）
```

### 4.3 优先级与 SLA

| 优先级 | SLA 时限 | 自动判定规则 |
|--------|---------|------------|
| P0 | 30 分钟 | 手动设定 |
| P1 | 2 小时 | 硬件故障、网络问题 |
| P2 | 8 小时 | 其他问题（默认） |
| P3 | 8 小时 | 权限申请 |

可在工单详情页手动调整优先级。超时工单会在管理后台顶栏弹出实时告警通知。

## 5. 会话管理

入口：**会话管理**

### 5.1 界面布局

```
┌──────────┬──────────────────────────┐
│ 待接入    │                          │
│ user-4-… │     聊天区域              │
│ 张三      │                          │
│          │  用户: 我的电脑蓝屏了      │
│ 我的会话  │  坐席: 请尝试重启...      │
│ user-2-… │                          │
│ 李四      │                          │
│          │  [快捷回复] [输入框] [发送] │
└──────────┴──────────────────────────┘
```

### 5.2 处理会话

1. 左侧「待接入」列表显示等待人工服务的用户
2. 点击「接入」开始服务
3. 在右侧聊天区与用户实时文字沟通
4. 处理完毕点击「关闭会话」，用户将收到服务评价邀请

### 5.3 快捷回复

在「系统设置 → 快捷回复」中预设常用话术。工作时点击即可一键发送。

| 示例 | 内容 |
|------|------|
| 问候语 | 您好，请问有什么可以帮您？ |
| 请稍候 | 请稍候，我帮您查询一下... |

## 6. 知识库管理

入口：**知识库管理**

### 6.1 文档分类

系统预设五个分类：

| 分类 | 内容示例 |
|------|---------|
| 人事制度 | 考勤、年假、报销制度 |
| IT 支持 | VPN 配置、邮箱设置、打印机安装 |
| 产品文档 | 产品手册、功能说明 |
| 培训资料 | 安全培训、入职培训 |
| 其他 | 通知公告等 |

### 6.2 上传文档

1. 点击「📎 上传文档」
2. 选择文件（支持 PDF / Word / TXT / Markdown）
3. 选择分类（可选，便于管理）
4. 添加标签（可选，逗号分隔，如：VPN,远程办公）
5. 点击上传

系统自动解析文档内容并建立向量索引，约 1-2 分钟后状态变为「就绪」。

### 6.3 管理文档

| 操作 | 说明 |
|------|------|
| 点击标题 | 查看文档详情（元数据） |
| 修改分类 | 列表中直接下拉选择新分类 |
| 删除 | 确认后移除（不可恢复） |
| 分类筛选 | 左侧分类栏点击切换 |

## 7. 系统设置

入口：**系统设置**

包含三个面板：用户管理、快捷回复、服务健康检查。

### 服务健康检查

检查各依赖服务的连通状态：

| 服务 | 用途 |
|------|------|
| Java Backend | 业务中台 |
| MySQL | 业务数据库 |
| Redis | 会话缓存 |
| Milvus | 向量检索 |
| MinIO | 文件存储 |

---

## 8. 坐席在线状态

登录管理后台后自动上线。点击顶栏「退出登录」自动下线。

> 在线坐席才能接收 SLA 告警通知和接入用户会话。
