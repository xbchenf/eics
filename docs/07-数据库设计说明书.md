# EICS 企业智能客服系统 — 数据库设计说明书

> 版本：V2.0 | 最后更新：2026-06-28

---

## 1. 概述

- 数据库：MySQL 8.0
- 引擎：InnoDB
- 字符集：utf8mb4
- 时区：Asia/Shanghai (+08:00)

## 3. 表结构详述

### 3.1 sys_user — 系统用户表

统一管理普通用户、坐席、管理员。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 用户 ID |
| username | VARCHAR(50) | UNIQUE, NOT NULL | 登录用户名 |
| password_hash | VARCHAR(255) | NOT NULL | BCrypt 加密 |
| name | VARCHAR(100) | DEFAULT '' | 显示姓名 |
| phone | VARCHAR(20) | DEFAULT '' | 手机号 |
| role | VARCHAR(20) | DEFAULT 'USER' | USER / AGENT / ADMIN |
| status | VARCHAR(20) | DEFAULT 'ACTIVE' | ACTIVE / DISABLED |
| create_time | DATETIME | DEFAULT NOW() | 创建时间 |
| update_time | DATETIME | ON UPDATE NOW() | 更新时间 |

### 3.2 cs_chat_session — 聊天会话表

记录每次用户接入的会话信息。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | VARCHAR(64) | PK | 会话 ID（格式：user-{userId}-{timestamp}） |
| user_id | VARCHAR(100) | DEFAULT '' | 用户标识 |
| status | VARCHAR(20) | DEFAULT 'BOT' | BOT / WAITING / AGENT / CLOSED |
| agent_id | BIGINT | NULL | 分配的坐席 ID |
| create_time | DATETIME | DEFAULT NOW() | 创建时间 |
| update_time | DATETIME | ON UPDATE NOW() | 更新时间 |
| close_time | DATETIME | NULL | 关闭时间 |

索引：`idx_status`, `idx_user_id`

### 3.3 cs_chat_message — 聊天消息表

记录会话中的每一条消息。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 消息 ID |
| session_id | VARCHAR(64) | NOT NULL | 所属会话 ID |
| sender_type | VARCHAR(20) | NOT NULL | USER / BOT / AGENT |
| content | MEDIUMTEXT | NOT NULL | 消息内容 |
| msg_type | VARCHAR(20) | DEFAULT 'TEXT' | TEXT / IMAGE / FILE |
| metadata | TEXT | NULL | 附加元数据 JSON |
| create_time | DATETIME | DEFAULT NOW() | 发送时间 |

索引：`idx_session_id`, `idx_create_time`

### 3.4 cs_service_order — 客服工单表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | VARCHAR(64) | PK | 工单 ID（UUID） |
| session_id | VARCHAR(64) | NOT NULL | 关联会话 ID |
| phone | VARCHAR(20) | DEFAULT '' | 用户手机号 |
| fault_description | TEXT | NOT NULL | 故障描述 |
| device_id | VARCHAR(200) | DEFAULT '' | 设备编号 |
| issue_type | VARCHAR(50) | DEFAULT 'OTHER' | 硬件故障/网络问题/权限申请/软件安装/其他 |
| status | VARCHAR(20) | DEFAULT 'PENDING' | PENDING / PROCESSING / RESOLVED / CLOSED |
| priority | VARCHAR(10) | DEFAULT 'P2' | P0 / P1 / P2 / P3 |
| sla_deadline | DATETIME | NULL | SLA 截止时间 |
| first_response_time | DATETIME | NULL | 首次响应时间（认领时记录） |
| agent_id | BIGINT | NULL | 受理坐席 ID |
| create_time | DATETIME | DEFAULT NOW() | 创建时间 |
| update_time | DATETIME | ON UPDATE NOW() | 更新时间 |
| resolve_time | DATETIME | NULL | 解决时间 |

索引：`idx_session_id`, `idx_status`, `idx_priority`

**优先级自动判定**：
- 硬件故障 / 网络问题 → P1 (SLA 2h)
- 权限申请 → P3 (SLA 8h)
- 其他 → P2 (SLA 8h)
- P0 仅手动设定 (SLA 30min)

### 3.5 cs_satisfaction — 满意度评价表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 评价 ID |
| session_id | VARCHAR(64) | UNIQUE, NOT NULL | 关联会话 ID（一会话一评价） |
| order_id | BIGINT | NULL | 关联工单 ID |
| user_id | BIGINT | NOT NULL | 评价用户 ID |
| agent_id | BIGINT | NULL | 被评价坐席 ID |
| rating | TINYINT | NOT NULL | 评分 1-5 |
| comment | VARCHAR(500) | NULL | 评价留言 |
| source | VARCHAR(20) | DEFAULT 'SESSION' | SESSION / ORDER / MANUAL |
| create_time | DATETIME | DEFAULT NOW() | 评价时间 |

索引：`idx_session` (UNIQUE), `idx_agent`, `idx_create_time`

### 3.6 cs_quick_reply — 快捷回复表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 回复 ID |
| agent_id | BIGINT | NULL | 所属坐席 ID（NULL=公用） |
| title | VARCHAR(100) | NOT NULL | 按钮标题 |
| content | TEXT | NOT NULL | 回复内容 |
| category | VARCHAR(50) | DEFAULT '其他' | 分组 |
| sort_order | INT | DEFAULT 0 | 排序 |
| create_time | DATETIME | DEFAULT NOW() | 创建时间 |

### 3.7 ek_document — 知识库文档表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 文档 ID |
| title | VARCHAR(500) | NOT NULL | 文档标题 |
| file_name | VARCHAR(500) | NOT NULL | 原始文件名 |
| file_type | VARCHAR(50) | NOT NULL | pdf / docx / txt / md |
| file_size | BIGINT | DEFAULT 0 | 文件大小（字节） |
| minio_path | VARCHAR(1000) | DEFAULT '' | MinIO 存储路径 |
| status | VARCHAR(20) | DEFAULT 'PENDING' | PENDING / PARSING / READY / FAILED |
| category | VARCHAR(50) | NULL | 人事制度/IT支持/产品文档/培训资料/其他 |
| tags | VARCHAR(500) | NULL | 标签（逗号分隔） |
| chunk_count | INT | DEFAULT 0 | 切片总数 |
| deleted | TINYINT | DEFAULT 0 | 逻辑删除 |
| create_time | DATETIME | DEFAULT NOW() | 上传时间 |
| update_time | DATETIME | ON UPDATE NOW() | 更新时间 |

索引：`idx_status`, `idx_create_time`

### 3.8 ek_document_chunk — 文档切片表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 切片 ID |
| document_id | BIGINT | NOT NULL | 所属文档 ID |
| chunk_index | INT | NOT NULL | 切片序号（从 0 开始） |
| content | MEDIUMTEXT | NOT NULL | 切片文本内容 |
| char_count | INT | DEFAULT 0 | 字符数 |
| milvus_id | BIGINT | NULL | Milvus 向量 ID |
| create_time | DATETIME | DEFAULT NOW() | 创建时间 |

索引：`idx_document_id`, `idx_milvus_id`

## 4. 索引策略

| 表 | 索引 | 类型 | 用途 |
|------|------|------|------|
| sys_user | idx_username | UNIQUE | 登录查询 |
| sys_user | idx_role | NORMAL | 角色筛选 |
| cs_chat_session | idx_status | NORMAL | 按状态查询（WAITING 队列） |
| cs_chat_session | idx_user_id | NORMAL | 按用户查询会话 |
| cs_chat_message | idx_session_id | NORMAL | 查询会话消息 |
| cs_service_order | idx_status | NORMAL | 按状态筛选 |
| cs_service_order | idx_priority | NORMAL | SLA 监控查询 |
| cs_satisfaction | idx_session | UNIQUE | 防重复评价 |
| cs_satisfaction | idx_agent | NORMAL | 坐席绩效统计 |
| ek_document | idx_status | NORMAL | 就绪文档筛选 |
| ek_document_chunk | idx_document_id | NORMAL | 文档切片关联 |

## 5. 数据生命周期

| 数据 | 保留策略 |
|------|---------|
| 聊天消息 | 永久保留（用于历史回溯） |
| 工单 | 永久保留 |
| 评价 | 永久保留 |
| 会话缓存 (Redis) | 30 分钟 TTL 自动过期 |
| 坐席在线状态 (Redis) | Session 级别 |
| 上传文档 | 逻辑删除，不物理清除 |
