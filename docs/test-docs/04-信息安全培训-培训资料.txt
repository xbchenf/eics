信息安全培训手册（2026年度）

一、密码安全
1. 密码长度至少 8 位，包含大小写字母、数字、特殊字符
2. 每 90 天更换一次密码
3. 不同系统使用不同密码，禁止重复使用
4. 禁止将密码写在便签纸上或共享给他人

二、数据分级保护
公司将数据分为三个等级：
1. 公开数据：公司公告、产品宣传资料
2. 内部数据：员工通讯录、部门文档
3. 机密数据：客户信息、财务报表、源代码

机密数据必须加密存储，传输必须使用加密通道。

三、邮件安全
1. 不点击来源不明的邮件附件和链接
2. 发件人可疑时，先电话确认再处理
3. 涉及敏感信息的邮件使用加密发送
4. 定期清理邮箱，删除垃圾邮件

四、移动设备安全
1. 公司配发的笔记本电脑必须安装公司指定的杀毒软件
2. 个人手机连接公司 WiFi 需先注册 MAC 地址
3. 丢失设备立即报告 IT 部门进行远程擦除
4. USB 移动存储设备使用前必须扫描病毒

五、安全事件报告
发现安全事件（病毒、数据泄露、异常登录）应立即：
1. 断网隔离受影响的设备
2. 报告 IT 安全团队（分机：8002）
3. 保留现场，不要自行处理
