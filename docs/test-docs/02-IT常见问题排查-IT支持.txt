IT 常见问题排查手册

一、VPN 连接问题
症状：VPN 客户端提示"连接失败"或"认证失败"
排查步骤：
1. 确认网络连接正常，可以访问外网
2. 检查 VPN 客户端版本是否为最新版（v3.2.1+）
3. 确认账号密码是否正确（区分大小写）
4. 尝试切换 VPN 节点（北京/上海/广州）
5. 如仍无法连接，提交 IT 工单，IT 将在 2 小时内响应

二、电脑蓝屏问题
症状：Windows 突然蓝屏，错误代码如 0x0000007B
常见原因及处理：
1. 驱动不兼容 → 更新显卡/主板驱动到最新版
2. 内存故障 → 重新插拔内存条，清理金手指
3. 系统文件损坏 → 以管理员身份运行 sfc /scannow
4. 硬盘故障 → 运行 chkdsk /f 检查磁盘
5. 频繁蓝屏 → 提交 IT 报修工单，工程师上门检测

三、邮箱配置
企业邮箱地址格式：姓名全拼@company.com
客户端配置：
- IMAP 服务器：imap.company.com，端口 993，SSL
- SMTP 服务器：smtp.company.com，端口 465，SSL
- 网页版：http://mail.company.com

四、打印机使用
公司打印机 IP 地址：192.168.1.50
安装步骤：
1. 控制面板 → 设备和打印机 → 添加打印机
2. 选择"使用 IP 地址添加"
3. 输入 192.168.1.50，自动安装驱动
4. 墨盒更换：提交 IT 工单 → IT 上门更换
